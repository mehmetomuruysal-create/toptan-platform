# ToplAl — Toplu Satın Alma Platformu

## Proje Hakkında
Bireysel, KOBİ ve Kurumsal kullanıcıların toplu alım ihalelerine katılabildiği, 
tedarikçilerle buluştuğu B2B/B2C e-ticaret platformu.

## Dosyalar
- `index.html` → Tam çalışan interaktif prototip (tek dosya, kurulum gerektirmez)
- `TopluSatinAlma_YolHaritasi.docx` → Detaylı iş planı ve yol haritası
- `README.md` → Bu dosya

## Prototipi Çalıştırma
`index.html` dosyasını çift tıklayarak herhangi bir tarayıcıda açabilirsiniz.
Kurulum veya sunucu gerekmez.

## Özellikler (Prototip)
- Ana sayfa: Aktif ihaleler, arama, filtreler
- İhale detay sayfası: Fiyat eşikleri, kargo bilgisi, katılımcılar
- İhale oluşturma: 4 adımlı form, hedef kitle ve minimum miktar tanımlama
- Konum seçimi: Ülke / Şehir / İlçe (TR, DE, UK, US, AE, NL)
- Üye ol: 3 klasman (Bireysel, KOBİ, Kurumsal)
- Giriş yap modal
- Kategoriler modal (10 kategori)
